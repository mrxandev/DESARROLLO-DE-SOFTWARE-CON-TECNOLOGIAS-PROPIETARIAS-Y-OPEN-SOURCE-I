# Sistema de Ticket

## Descripción
Proyecto colaborativo para la gestión de tickets de soporte técnico.

---

## Caso de Uso – Sistema de Soporte Técnico

### Problema
Los usuarios reportan problemas técnicos por mensajes o llamadas, lo que provoca desorden y retrasos en la solución de incidencias.

### Objetivo
Permitir registrar y gestionar tickets de soporte técnico.

### Actores
- Usuario
- Técnico de soporte
- Administrador

### Flujo Principal
1. El usuario crea un ticket.
2. El sistema registra la incidencia.
3. El técnico revisa el ticket.
4. El técnico cambia el estado del caso.
5. El usuario recibe seguimiento.

### Problemas comunes
- Internet no funciona
- PC lenta
- Impresora dañada
- Contraseña olvidada
